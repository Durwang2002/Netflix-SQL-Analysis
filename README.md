# 🎬 Netflix SQL Data Analysis Project

## 📌 Project Overview

This project analyzes the Netflix Titles dataset using SQL to uncover meaningful business insights and demonstrate end-to-end Data Analyst skills. The analysis follows a structured analytics workflow covering database understanding, data quality assessment, exploratory analysis, KPI reporting, advanced SQL techniques, and business recommendations.

The project is designed to simulate a real-world business analysis scenario and showcase SQL proficiency required for Data Analyst roles.

# 📈 Key Business Insights

- Netflix catalog is dominated by one content type.
- Content acquisition increased significantly in recent years.
- A few countries contribute the majority of available titles.
- Metadata quality can be improved by filling missing values.
- Most content belongs to recent release years.
- Content acquisition often occurs years after the original release.
- Movies generally fall within a standard duration range.
- TV Shows are primarily short to medium length.

---

# 💡 Recommendations

- Improve metadata completeness.
- Expand content acquisition from underrepresented countries.
- Reduce acquisition delays.
- Invest in high-performing directors.
- Maintain a balanced content portfolio.
- Build BI dashboards for monitoring KPIs.
- Implement automated data quality checks.
